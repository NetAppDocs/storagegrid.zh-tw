= ECMC ：遺失抹除編碼資料片段
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
此稽核訊息表示系統偵測到遺失的銷毀編碼資料片段。

[cols="1a,1a,4a"]
|===
| 程式碼 | 欄位 | 說明 


 a| 
VCMC
 a| 
VCS ID
 a| 
包含遺失區塊的VCS名稱。



 a| 
MCID
 a| 
區塊ID
 a| 
遺失銷毀編碼片段的識別碼。



 a| 
RSRLT
 a| 
結果
 a| 
此欄位的值為「無」。RSRLT為必填訊息欄位、但與此特定訊息無關。使用「無」而非「UCS」、因此不會篩選此訊息。

|===
