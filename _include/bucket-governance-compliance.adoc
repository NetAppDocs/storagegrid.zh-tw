| 預設保留模式 | 說明 


 a| 
治理
 a| 
* 擁有權限的使用者 `s3:BypassGovernanceRetention`可以使用 `x-amz-bypass-governance-retention: true`要求標頭來略過保留設定。
* 這些使用者可以在達到物件版本的保留截止日期之前刪除物件版本。
* 這些使用者可以增加、減少或移除物件的保留到目前為止。




 a| 
法規遵循
 a| 
* 直到達到物件的保留日期、才能刪除物件。
* 物件的保留日期可以增加、但不能減少。
* 直到達到該日期為止、才能移除物件的保留日期。


* 注意 * ：您的網格管理員必須允許您使用法規遵循模式。
